為什麼是
「極度輕量化」

框架滿街跑，為何我們比較優秀？

市面上的 Web 框架很多，支援 Java 語言的也滿山滿海。但他們都有一個通病，就是為了彈性、為了高擴充性，而造成框架肥大、不易快速開發、不易理解。

Aok 則不一樣，我們的出發點是希望開發者能「花一個下午可以寫一個班級管理系統」出來（相信我，我們測試過了，沒問題）。我們希望開發者可以快速上手這個框架，並且可以花很短的時間，就能開發出屬於自己的API服務、動態網頁。如此一來，開發者可以更專注在使用者需求上，而不是框架怎麼用。

框架特色

給你用 Aok 的四個裡由

框架程式碼輕量

少的不可思議，框架維護成本低

如此一來，未來如果有什麼 Bugs 或者需要新增功能、擴充功能。都可以很輕易的更新框架。甚至，你也可以一起來維護

簡易上手

一個下午就學會框架全部

一般學習新框架，要花你多少時間？三天？一個星期？一個月？我可以跟你保證，學習 Aok，你只需要一個下午，甚至更短，就可以成為 Aok 高手了

快速開發

一個下午開發班級管理系統真心不騙

經過我們的實測，只要充份使用框架，一個下午就可以開發一個班級管理系統，外加部署。讓開發者少一點時間寫程式，多一點時間給創意、需求

MVC架構

Model、View、Controller 多人開發更好用

大學專題需要多人開發？別擔心，使用 Aok，你可以放心的多人開發，將前端、後端的部分完全分開、不互相干擾。未來要整合也相當容易，git merge 就完成

我 OK，你先 Aok

你猶豫要不要用 Aok 的時間，已經足夠學會它了
