AokRouter

給你一條通往成功的路

主要負責處理靜態檔案，檔案路徑預設在該project下層的資料夾下(AoK/src/Assets/檔案名稱)，像是html、css、jpg….等型態的檔案。

方法詳細定義：

方法 說明
handle(HttpExchange exchange) 為實作 HttpHandler 的方法，當路由接收到請求的處理演算法在此方法中定義。其中使用 AokController 作為 Stretagy。其中的 HTTP Method 演算法由 AokController 提供。
setController(AokController ctr) 設定 Controller 屬性。
setContentType() 根據 response 屬性，可以知道此次返回的 Content Type 為何，並且賦予適當的header，讓Server可以成功執行。
