AokController

重新掌握你的框架

主要負責處理 GET 、POST 、PUT 、DELETE ，這些 HttpMethod被 client 端呼叫時所需做的動作。

建構子詳細定義：

建構子 說明
AokController(AokModel model, AokView view) 給定view和model初值。

方法詳細定義：

方法 說明
Get()、Post()、Delete()、Put() 處理被呼叫時須執行的動作，使用者可以覆寫此 function 。
setView(AokView v)、setModel(AokModel m) 設定 view 、 model 的值。
